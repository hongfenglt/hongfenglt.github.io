轻量级的 UI 实时调试工具 - Sherlock

这是除 Reveal 外又一款支持实时调试 UI 的工具，相比于 Reveal，Sherlock 虽然没有 Reveal 那么强大的功能，但它更加轻量级，且无需任何配置，打开模拟器和 Sherlock 就可以直接使用。此外，它还支持调整屏幕大小以测试你的 UI 在各类屏幕上的展示效果，不需要你去启动各种模拟器。目前 Sherlock 正处于 Beta 测试中，暂时只支持 iPhone 模拟器。

  • 查看视图层级，定位视图

  • 直接修改视图各个属性，不需要重新编译

  • 查看视图位置

下载地址：https://sherlock.inspiredcode.io
目前是测试版，点击下载之后填写邮箱，会收到测试版的 License。

安装之后使用 Xcode 运行模拟器， 之后会在Sherlock显示正在活跃的 App ，command +Ｅ　激活／取消　Sherlock的活跃状态。
不需要任何配置就可以使用。
